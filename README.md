# 123go Event Feedback Form

A simple, responsive feedback form for the "123go" event.

## Deployment

This project is configured to deploy automatically to GitHub Pages using GitHub Actions.

### How to ensure it's live:
1. Go to the **Settings** tab of this repository.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment > Source**, ensure it is set to **GitHub Actions**.
4. Once set, the deployment will trigger automatically on every push to the `main` branch.

**Live URL:** [https://anirudhkotla.github.io/123go-feedback-form/](https://anirudhkotla.github.io/123go-feedback-form/)
